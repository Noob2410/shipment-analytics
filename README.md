# Версия auto-row

Сделано:
- Кнопка + Курьер полностью убрана.
- Новая строка курьера создается автоматически после полного заполнения последней строки маршрута.
- Автопереход курсора:
  прибытие часы → прибытие минуты → убытие часы → убытие минуты → следующая строка.
- Очистка конкретного маршрута за сегодня.
- Безопасное удаление: заполненную строку удаляет только после подтверждения.
- Экспорт CSV для Excel.
- Сохранена рабочая логика времени, расчетов, KPI, аналитики и логов.
