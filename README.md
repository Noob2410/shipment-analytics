# Версия с раздельным вводом часов и минут

Что исправлено:
- Время вводится через отдельные поля HH и MM.
- Часы ограничены 00–23.
- Минуты ограничены 00–59.
- После ввода двух цифр часов курсор переходит на минуты.
- Время сохраняется сразу и не исчезает.
- Расчет времени, KPI, среднее по маршруту, общая статистика и аналитика подтягиваются из сохраненных данных.
- Удаление строки и добавление курьера работают.
- Название маршрута кликабельно и подчеркивается при наведении.

Важно: после обновления на GitHub откройте сайт через Ctrl+F5 или в инкогнито.
