Как запустить через GitHub Pages:
1. Создай новый репозиторий на GitHub.
2. Загрузи index.html в корень репозитория.
3. Settings -> Pages -> Deploy from a branch.
4. Branch: main, папка /root.
5. Через 1-3 минуты сайт будет доступен.

Как пользоваться:
- Дата берется автоматически с устройства.
- По каждому маршруту вводится только время прибытия и убытия.
- Строки добавляются кнопкой + Курьер.
- Название маршрута ведет в аналитику маршрута.
- Кнопка Общая статистика ведет в общую аналитику.
- Данные хранятся в браузере на устройстве через localStorage.
- Для переноса данных есть Скачать данные JSON / Загрузить JSON.
